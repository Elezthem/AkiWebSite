Aki — мощный инструмент для вашего сервера.

Он может помочь вам с модерацией, а также сделать ваш сервер более интересным и увлекательным.

Добавить в Discord

Информация

У меня есть множество информационных команд. Например, для просмотра информации о сервере, пользователях, каналах и т.д.

Модерация

У меня есть полезные команды для модерации сервера. Например, для выдачи тайм-аута, киков, банов и обратных действий.

Утилиты

У меня есть множество полезных утилит. Например, для просмотра информации о игровом сервере CS:GO, Minecraft или другой игре, курсе валюты/криптовалюты, переворота текста, получения рандомного числа или просмотр аватара другого пользователя

Веселье

У меня есть несколько веселых команд. Например, вы можете погадать о судьбе с помощью "Шара судьбы "или подбросить монетку, чтобы решить вопрос.

Трекеры

У меня есть возможность установки "трекеров ". Например, вы можете установить трекер для отслеживания вашего игрового сервера.

Настройки

С помощью настроек, вы можете настроить удобный вам префикс, выбрать нужный язык, а так же установить те или иные настройки логирования.
